Quantum Physics Schr√∂dinger Wave Function Uncertainty

Introduction

Quantum mechanics revolutionized our understanding of nature at the atomic and subatomic scale. It replaces the deterministic classical worldview with probabilistic descriptions, introducing fundamental concepts like wave-particle duality and quantum superposition.

Postulates of Quantum Mechanics

Postulate 1: State Description

The state of a quantum system is completely described by a wave function $\Psi(\mathbf{r}, t)$ such that $|\Psi(\mathbf{r}, t)|^2 d\tau$ gives the probability of finding the particle in volume element $d\tau$.

Normalization condition:

$$\int_{all\ space} |\Psi(\mathbf{r}, t)|^2 d\tau = 1$$

Postulate 2: Observables

To every observable physical quantity corresponds a Hermitian operator. For example:

  • Position: $\hat{x} = x$
  • Momentum: $\hat{p}_x = -i\hbar \frac{\partial}{\partial x}$
  • Energy: $\hat{H} = -\frac{\hbar^2}{2m}\nabla^2 + V(\mathbf{r})$

Postulate 3: Measurement

The only possible results of measuring observable $\hat{A}$ are the eigenvalues $a_n$ of the equation:

$$\hat{A}\psi_n = a_n \psi_n$$

Postulate 4: Expectation Values

The expectation value of observable $\hat{A}$ in state $\Psi$ is:

$$\langle A \rangle = \langle \Psi | \hat{A} | \Psi \rangle = \int \Psi^* \hat{A} \Psi \, d\tau$$

Postulate 5: Time Evolution

The time evolution of a quantum system is governed by the time-dependent Schr√∂dinger equation:

$$i\hbar \frac{\partial \Psi}{\partial t} = \hat{H} \Psi$$

The Schr√∂dinger Equation

Time-Independent Schr√∂dinger Equation

For systems with time-independent Hamiltonians, we can separate variables:

$$\Psi(\mathbf{r}, t) = \psi(\mathbf{r}) e^{-iEt/\hbar}$$

Leading to the time-independent equation:

$$\hat{H}\psi = E\psi$$

This is an eigenvalue equation where $E$ represents the energy eigenvalues.

Solutions in One Dimension

For a particle in a potential $V(x)$:

$$-\frac{\hbar^2}{2m}\frac{d^2\psi}{dx^2} + V(x)\psi = E\psi$$

The general form can be written as:

$$\frac{d^2\psi}{dx^2} = -\frac{2m}{\hbar^2}[E - V(x)]\psi = -k^2(x)\psi$$

where $k(x) = \sqrt{2m[E-V(x)]/\hbar^2}$ is the local wave number.

Fundamental Quantum Systems

Particle in a Box

For infinite potential walls at $x = 0$ and $x = L$:

$$\psi_n(x) = \sqrt{\frac{2}{L}} \sin\left(\frac{n\pi x}{L}\right)$$ $$E_n = \frac{n^2 \pi^2 \hbar^2}{2mL^2}, \quad n = 1, 2, 3, \ldots$$

This demonstrates energy quantization and zero-point energy.

Quantum Harmonic Oscillator

For potential $V(x) = \frac{1}{2}m\omega^2 x^2$:

$$E_n = \hbar\omega\left(n + \frac{1}{2}\right), \quad n = 0, 1, 2, \ldots$$

The ground state wave function is:

$$\psi_0(x) = \left(\frac{m\omega}{\pi\hbar}\right)^{1/4} \exp\left(-\frac{m\omega x^2}{2\hbar}\right)$$

Hydrogen Atom

The hydrogen atom solution involves spherical coordinates and yields:

$$E_n = -\frac{13.6 \text{ eV}}{n^2}, \quad n = 1, 2, 3, \ldots$$

The wave function factorizes as:

$$\psi_{nlm}(r,\theta,\phi) = R_{nl}(r) Y_l^m(\theta,\phi)$$

where $R_{nl}(r)$ are radial functions and $Y_l^m(\theta,\phi)$ are spherical harmonics.

Heisenberg Uncertainty Principle

General Form

For any two observables $\hat{A}$ and $\hat{B}$:

$$\sigma_A \sigma_B \geq \frac{1}{2}|\langle[\hat{A}, \hat{B}]\rangle|$$

where $\sigma_A = \sqrt{\langle A^2 \rangle - \langle A \rangle^2}$ is the standard deviation.

Position-Momentum Uncertainty

The most famous case with $[\hat{x}, \hat{p}] = i\hbar$:

$$\sigma_x \sigma_p \geq \frac{\hbar}{2}$$

This fundamental limit reflects the wave-particle duality of quantum objects.

Energy-Time Uncertainty

For systems with finite lifetimes:

$$\sigma_E \sigma_t \geq \frac{\hbar}{2}$$

This explains phenomena like natural line widths and virtual particles.

Angular Momentum

Orbital Angular Momentum

The angular momentum operators satisfy:

$$[\hat{L}_i, \hat{L}_j] = i\hbar \epsilon_{ijk} \hat{L}_k$$

With eigenvalue equations:

$$\hat{L}^2 Y_l^m = \hbar^2 l(l+1) Y_l^m$$ $$\hat{L}_z Y_l^m = \hbar m Y_l^m$$

where $l = 0, 1, 2, \ldots$ and $m = -l, -l+1, \ldots, l-1, l$.

Spin Angular Momentum

Intrinsic angular momentum with similar commutation relations:

$$\hat{S}^2 |s, m_s\rangle = \hbar^2 s(s+1) |s, m_s\rangle$$ $$\hat{S}_z |s, m_s\rangle = \hbar m_s |s, m_s\rangle$$

For spin-1/2 particles (electrons, protons): $s = 1/2$, $m_s = \pm 1/2$.

Computational Examples

Here's a Python simulation of wave packet evolution demonstrating quantum spreading: 

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

def gaussian_wave_packet(x, t, x0, k0, sigma, m=1, hbar=1):
    """
    Time evolution of a Gaussian wave packet
    """
    # Time-dependent width
    sigma_t = sigma * np.sqrt(1 + (hbar * t / (2 * m * sigma**2))**2)
    
    # Phase terms
    phase = k0 * (x - x0) - (hbar * k0**2 * t) / (2 * m)
    phase += (x - x0 - hbar * k0 * t / m)**2 / (4 * sigma_t**2)
    
    # Amplitude
    amplitude = (2 * np.pi * sigma_t**2)**(-1/4)
    
    # Wave function
    psi = amplitude * np.exp(-((x - x0 - hbar * k0 * t / m)**2) / (4 * sigma_t**2)) * np.exp(1j * phase)
    
    return psi, sigma_t

# Parameters
L = 20.0           # Box length
N = 1000           # Number of points
x = np.linspace(-L/2, L/2, N)
dx = x[1] - x[0]

# Initial wave packet parameters
x0 = -5.0          # Initial position
k0 = 2.0           # Initial momentum (in units of hbar)
sigma0 = 0.5       # Initial width
m = 1.0            # Mass
hbar = 1.0         # Reduced Planck constant

# Time parameters
dt = 0.01
t_max = 5.0
times = np.arange(0, t_max, dt)

# Create figure
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(12, 8))

def animate(frame):
    ax1.clear()
    ax2.clear()
    
    t = times[frame]
    psi, sigma_t = gaussian_wave_packet(x, t, x0, k0, sigma0, m, hbar)
    
    # Probability density
    prob_density = np.abs(psi)**2
    
    # Plot wave function (real and imaginary parts)
    ax1.plot(x, np.real(psi), 'b-', label='Re(œà)', linewidth=2)
    ax1.plot(x, np.imag(psi), 'r-', label='Im(œà)', linewidth=2)
    ax1.plot(x, prob_density, 'k-', label='|œà|¬≤', linewidth=2)
    
    ax1.set_xlabel('Position x')
    ax1.set_ylabel('Wave function')
    ax1.set_title(f'Quantum Wave Packet Evolution (t = {t:.2f})')
    ax1.legend()
    ax1.grid(True)
    ax1.set_ylim(-0.8, 0.8)
    
    # Plot uncertainty evolution

Quantum Tunneling

Transmission Through a Barrier

For a rectangular potential barrier of height $V_0$ and width $a$:

$$T = \frac{1}{1 + \frac{V_0^2 \sinh^2(\kappa a)}{4E(V_0 - E)}}$$

where $\kappa = \sqrt{2m(V_0 - E)}/\hbar$ for $E < V_0$.

In the limit of a thick, high barrier:

$$T \approx e^{-2\kappa a} = \exp\left(-2a\sqrt{\frac{2m(V_0 - E)}{\hbar^2}}\right)$$

Applications

  • Scanning Tunneling Microscopy (STM): Atomic-resolution imaging
  • Tunnel Diodes: Negative resistance devices
  • Nuclear Fusion: Coulomb barrier penetration in stars
  • Alpha Decay: Nuclear tunneling process

Many-Body Quantum Systems

Identical Particles

For a system of $N$ identical particles, the wave function must be either:

  • Symmetric (Bosons): $\Psi(1,2,\ldots,N) = \Psi(2,1,\ldots,N)$
  • Antisymmetric (Fermions): $\Psi(1,2,\ldots,N) = -\Psi(2,1,\ldots,N)$

Pauli Exclusion Principle

No two fermions can occupy the same quantum state:

$$\Psi(1,2,\ldots,N) = \frac{1}{\sqrt{N!}} \det \begin{pmatrix} \psi_1(\mathbf{r}_1) & \psi_1(\mathbf{r}_2) & \cdots \\ \psi_2(\mathbf{r}_1) & \psi_2(\mathbf{r}_2) & \cdots \\ \vdots & \vdots & \ddots \end{pmatrix}$$

This is the Slater determinant representation.

Entanglement

For composite systems, states that cannot be written as products:

$$|\Psi\rangle \neq |\psi_A\rangle \otimes |\psi_B\rangle$$

Example: Bell states

$$|\Phi^+\rangle = \frac{1}{\sqrt{2}}(|00\rangle + |11\rangle)$$

Summary

Quantum mechanics fundamentally changes our understanding of reality:

  • Probabilistic Nature: Measurements yield probabilistic outcomes
  • Wave-Particle Duality: All particles exhibit both wave and particle characteristics
  • Uncertainty Principle: Fundamental limits on simultaneous knowledge of observables
  • Superposition: Quantum systems can exist in multiple states simultaneously
  • Entanglement: Non-local correlations between separated systems

These principles enable technologies like lasers, transistors, MRI machines, and form the foundation for emerging quantum technologies including quantum computing and quantum cryptography.
