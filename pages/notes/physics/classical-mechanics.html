Classical Mechanics

Physics June 19, 2025
Mechanics Lagrangian Hamiltonian Newtonian

Introduction

Classical mechanics describes the motion of macroscopic objects, from projectiles to planets, using deterministic laws. This foundational theory provides three equivalent formulations: Newtonian, Lagrangian, and Hamiltonian mechanics.

Newtonian Mechanics

Newton's Laws

The foundation of classical mechanics rests on Newton's three laws:

  1. First Law (Inertia): A body at rest stays at rest, and a body in motion stays in motion, unless acted upon by a net external force.
  2. Second Law: The acceleration of an object is directly proportional to the net force acting on it.
  3. Third Law: For every action, there is an equal and opposite reaction.

Equation of Motion

Newton's second law in its most general form:

$$\mathbf{F} = \frac{d\mathbf{p}}{dt} = \frac{d(m\mathbf{v})}{dt}$$

For constant mass, this reduces to the familiar:

$$\mathbf{F} = m\mathbf{a} = m\frac{d^2\mathbf{r}}{dt^2}$$

Central Force Problems

For a central force $\mathbf{F} = f(r)\hat{\mathbf{r}}$, we can derive the effective potential:

$$U_{\text{eff}}(r) = U(r) + \frac{L^2}{2mr^2}$$

where $L$ is the conserved angular momentum and the second term is the centrifugal barrier.

Lagrangian Mechanics

The Lagrangian Function

The Lagrangian is defined as the difference between kinetic and potential energy:

$$\mathcal{L}(q_i, \dot{q}_i, t) = T - V$$

where $q_i$ are generalized coordinates and $\dot{q}_i$ are generalized velocities.

Euler-Lagrange Equations

The equations of motion are derived from Hamilton's principle of least action:

$$\frac{d}{dt}\left(\frac{\partial \mathcal{L}}{\partial \dot{q}_i}\right) - \frac{\partial \mathcal{L}}{\partial q_i} = 0$$

This elegant formulation automatically handles constraints and is coordinate-independent.

Conserved Quantities

Noether's theorem connects symmetries to conservation laws:

  • Time translation symmetry ‚Üí Energy conservation
  • Spatial translation symmetry ‚Üí Momentum conservation
  • Rotational symmetry ‚Üí Angular momentum conservation

If the Lagrangian doesn't explicitly depend on a coordinate $q_i$, then:

$$p_i = \frac{\partial \mathcal{L}}{\partial \dot{q}_i} = \text{constant}$$

Hamiltonian Mechanics

The Hamiltonian Function

The Hamiltonian is obtained via Legendre transformation:

$$\mathcal{H}(q_i, p_i, t) = \sum_i p_i \dot{q}_i - \mathcal{L}$$

where $p_i = \frac{\partial \mathcal{L}}{\partial \dot{q}_i}$ are the generalized momenta.

Hamilton's Equations

The equations of motion in Hamiltonian form are:

$$\dot{q}_i = \frac{\partial \mathcal{H}}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial \mathcal{H}}{\partial q_i}$$

These first-order differential equations preserve phase space volume (Liouville's theorem).

Canonical Transformations

Transformations that preserve the Hamiltonian structure satisfy:

$$\sum_i p_i dq_i - \mathcal{H} dt = \sum_i P_i dQ_i - \mathcal{K} dt + dF$$

where $F$ is a generating function and $(Q_i, P_i)$ are new canonical coordinates.

Key Applications

Simple Harmonic Oscillator

The paradigmatic system with Hamiltonian:

$$\mathcal{H} = \frac{p^2}{2m} + \frac{1}{2}kx^2$$

Leading to the familiar solution $x(t) = A\cos(\omega t + \phi)$ with $\omega = \sqrt{k/m}$.

Two-Body Problem

Reducible to a one-body problem in the center-of-mass frame with reduced mass:

$$\mu = \frac{m_1 m_2}{m_1 + m_2}$$

For gravitational interactions, this leads to Kepler's laws and conic section orbits.

Rigid Body Motion

The rotational kinetic energy in terms of angular velocity $\boldsymbol{\omega}$ and inertia tensor $\mathbf{I}$:

$$T_{\text{rot}} = \frac{1}{2}\boldsymbol{\omega} \cdot \mathbf{I} \cdot \boldsymbol{\omega}$$

Leading to Euler's equations for torque-free motion.

Computational Examples

Here's a Python implementation of a simple pendulum using the Lagrangian approach: 

import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import odeint

def pendulum_lagrangian(state, t, g, L):
    """
    Solve pendulum motion using Lagrangian mechanics
    state = [theta, theta_dot]
    """
    theta, theta_dot = state
    
    # Euler-Lagrange equation for simple pendulum
    # d/dt(‚àÇL/‚àÇŒ∏Ãá) - ‚àÇL/‚àÇŒ∏ = 0
    # gives: Œ∏Ãà + (g/L)sin(Œ∏) = 0
    theta_ddot = -(g/L) * np.sin(theta)
    
    return [theta_dot, theta_ddot]

# Parameters
g = 9.81  # gravity
L = 1.0   # length
theta0 = np.pi/3  # initial angle
omega0 = 0.0      # initial angular velocity

# Time array
t = np.linspace(0, 10, 1000)

# Solve ODE
initial_state = [theta0, omega0]
solution = odeint(pendulum_lagrangian, initial_state, t, args=(g, L))

theta = solution[:, 0]
omega = solution[:, 1]

# Plot results
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(10, 8))

ax1.plot(t, theta, 'b-', linewidth=2)
ax1.set_xlabel('Time (s)')
ax1.set_ylabel('Angle Œ∏ (rad)')
ax1.set_title('Pendulum Motion')
ax1.grid(True)

# Phase space plot
ax2.plot(theta, omega, 'r-', linewidth=1.5)
ax2.set_xlabel('Angle Œ∏ (rad)')
ax2.set_ylabel('Angular velocity œâ (rad/s)')
ax2.set_title('Phase Space')
ax2.grid(True)

plt.tight_layout()
plt.show()

# Energy conservation check
def energy(theta, omega, g, L):
    """Calculate total energy"""
    kinetic = 0.5 * L**2 * omega**2
    potential = g * L * (1 - np.cos(theta))
    return kinetic + potential

E = energy(theta, omega, g, L)
print(f"Energy conservation: ŒîE/E‚ÇÄ = {(E.max() - E.min())/E[0]:.2e}")

Advanced Topics

Action-Angle Variables

For integrable systems, we can find canonical coordinates where:

$$\mathcal{H} = \mathcal{H}(I_1, I_2, \ldots, I_n)$$

The action variables $I_i$ are adiabatic invariants, and the angle variables $\theta_i$ evolve linearly in time.

Perturbation Theory

For nearly integrable systems with Hamiltonian:

$$\mathcal{H} = \mathcal{H}_0(I) + \epsilon \mathcal{H}_1(I, \theta, t)$$

Canonical perturbation theory allows systematic analysis of small deviations from integrability.

Chaos and KAM Theory

The Kolmogorov-Arnold-Moser theorem describes the persistence of quasi-periodic motion under small perturbations, while chaos emerges in regions where resonance conditions are met.

Summary

Classical mechanics provides three equivalent but distinct perspectives on mechanical systems:

  • Newtonian: Direct force analysis, intuitive but limited to Cartesian coordinates
  • Lagrangian: Energy-based approach, handles constraints naturally
  • Hamiltonian: Phase space formulation, foundation for statistical mechanics and quantum mechanics

These formulations not only solve mechanical problems but also provide the conceptual foundation for field theory, quantum mechanics, and modern physics.
